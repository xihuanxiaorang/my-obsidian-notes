## 题目描述

给你一个链表的头节点 `head` 和一个整数 `val`，请你删除链表中所有满足 `Node.val == val` 的节点，并返回 **新的头节点**。

**示例 1：**

![](https://img.xiaorang.fun/202508192306142.png)

```text
输入：head = [1,2,6,3,4,5,6], val = 6
输出：[1,2,3,4,5]
```

**示例 2：**

```text
输入：head = [], val = 1
输出：[]
```

**示例 3：**

```text
输入：head = [7,7,7,7], val = 7
输出：[]
```

**提示：**

- 链表中的节点数目在范围 <code>[0, 10<sup>4</sup>]</code> 内
- `1 <= Node.val <= 50`
- `0 <= val <= 50`
