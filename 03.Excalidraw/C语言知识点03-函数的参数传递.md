函数的参数传递 ^YSPrhWJZ

存在两种方式：值传递和地址传递 ^NBw9BDQA

这种方式使用类型为整型、浮点型、字符型...的变量、常量、数组元素作为函数参数，实际上是将实参的值复制到形参相应的存储单元中，
即形参和实参分别占用不同的存储单元中，这种传递方式称为"参数的值传递"。 ^JyEe0pyM

值传递 ^ilI7PIcm

值传递的特点是单向传递，传递给形参的是实参的值，在函数被调用时才会给形参分配内存，调用结束后，就会释放内存。
形参值的任何变化并不会改变实参的值。 ^Nsd4OLE8

#include <stdio.h>

int main()
{
    int a = 3, b = 4;
    printf("swap函数调用前: a = %d，b = %d\n", a, b);
    swap(a, b);
    printf("swap函数调用后: a = %d，b = %d\n", a, b);
    return 0;
}

void swap(int m, int n)
{
    int temp = m;
    m = n;
    n = temp;
} ^MlT8RHVG

地址传递 ^g54PnUin

这种方式使用数组名或者指针作为函数参数，传递的是该数组的首地址或指针的值（指针所指向变量的地址），形参自然接收到的就是地址，
即形参与实参共用实参的存储空间，在函数调用过程中，并不会为形参开启新的存储空间，对形参的操作就是对实参本身的操作。 ^dBfEkYmw
